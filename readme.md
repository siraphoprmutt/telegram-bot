# Telegram Bot

## 1. ทำการ Download หรือ Clone Repo

- Clone repo

```bash

git clone https://github.com/siraphoprmutt/telegram-bot.git
cd telegram-bot

```

## 2. เปลี่ยนชื่อไฟล์ .env.exemple เป็น .env

- เปลี่ยน BOT_TOKEN ที่ไฟล์ .env

## 3. ติดตั้ง node_module และ run dev

```bash

npm install
npm run dev

```
